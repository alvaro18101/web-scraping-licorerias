# Web Scraping de Licorerías
Se extraen los datos de las siguientes licorerías:
- Almendariz: <https://almendariz.com.pe/>
- El Pozito: <https://elpozito.com.pe/el-pozito/inicio>
- Licorerías Unidas: <https://licoreriasunidas.pe/>

Los archivos serán exportados en un archivo Excel con los siguientes encabezados:
- Fecha de consulta
- Nombre del producto
- Precio normal
- Precio de oferta
- Cantidad
- Categoría

Instala las librerías requeridas en tu sistema o en un entorno virtual ejecutando en la consola:
```bash
pip install -r requirements.txt
```
